
En Özetle
**Managed** = runtime yönetiyor  
**Unmanaged** = yönetim daha çok sende

## Managed Code nedir?

**Managed code**, CLR (Common Language Runtime) kontrolünde çalışan koddur.

Managed code'da Runtimde de ne yapılıyor.
Yani .NET oramı  kodu 
- çalıştırır
- bellek yönetimini yapar
- Garbage Collection uygular
- tip güvenliği sağlar
- exception yönetimine destek olur
## Unmanaged Code nedir?

**Unmanaged code**, CLR dışında çalışan, belleği ve kaynakları daha doğrudan yöneten koddur.

Unmanaged code da 
- Bellek yönetimi developera aitir.
- GC doğrudan geçerli değildir. 
-  işletim sistemi  ile daha iç içedir.

Örnek diller; C++ , C , Assembly
